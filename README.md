# 📊 Student Performance Analysis using R

## 📌 Project Overview

This project analyzes the **Student Performance Dataset** from the UCI Machine Learning Repository using **R** and **R Markdown**. The goal is to explore the factors that influence students' academic performance through data visualization and statistical analysis.

---

## 🎯 Objectives

- Understand the characteristics of the student performance dataset.
- Perform data cleaning and preprocessing.
- Conduct exploratory data analysis (EDA).
- Visualize relationships between different variables.
- Identify factors affecting students' final grades.
- Generate a reproducible report using R Markdown.

---

## 📂 Dataset

**Source:** UCI Machine Learning Repository

**Dataset:** Student Performance Dataset

The dataset contains information about students such as:

- Gender
- Age
- Study Time
- Family Support
- Absences
- Internet Access
- Parents' Education
- Health Status
- Alcohol Consumption
- Final Grades (G1, G2, G3)

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/MukulBijalwan/Student_Performance_Analysis.git
```

2. Open `StudentPerformance.Rproj` in RStudio.

3. Install required packages:

```r
install.packages(c("ggplot2","dplyr","tidyverse","readr","corrplot","rmarkdown","psych"))
```

4. Open `analysis.Rmd`.

5. Click **Knit** to generate the report.

---

## 📌 Key Findings

- Study time has a positive relationship with student performance.
- Higher absenteeism is associated with lower final grades.
- Family support and parental education show noticeable influence on academic performance.
- Visualizations help identify important trends and patterns in the dataset.
